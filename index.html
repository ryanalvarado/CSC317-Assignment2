How to Apply CSS to HTML

Create a style.css file in the directory of your web page. In the <head> section of HTML document, paste this code in to reference your CSS code:

<link rel="stylesheet" href="style.css">

In your style.css you can then style your HTML. For example:

body {
        font-family: Arial, sans-serif;
        margin: 0px 135px 0px 135px;
        color: rgb(120, 120, 120);
      }

How to Use JavaScript in a Webpage

You can also link to a Javascript file that yout HTML page can reference. Create a Javscript file in your webpage's directory and add this in your HTML <head> tag:

<script src="script.js"></script>
